# Ralph Loop

Spec-first CLI for preparing worktrees, running the Ralph Wigum policy defined by Loop State Machine, and inspecting active Ralph Loop sessions.

Ralph Loop is not a standalone loop definition. It is a concrete CLI and repository contract layered on top of the Ralph policy from [`loop-state-machine.spec`](https://github.com/siisee11/loop-state-machine.spec).
