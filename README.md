# Ralph Loop

Spec-first CLI for preparing worktrees, running the Ralph Wigum policy defined by What The Loop, and inspecting active Ralph Loop sessions.

Ralph Loop is not a standalone loop definition. It is a concrete CLI and repository contract layered on top of the Ralph policy from [`what-the-loop.spec`](https://github.com/siisee11/what-the-loop.spec).
